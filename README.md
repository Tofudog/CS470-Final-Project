# CS470-Final-Project
This is a public repository containing the code and relevant files for our CS470 Data Mining Final Project.

### How to use this repository
1. Either fork this repo or click `download zip`
2. Download Anaconda Navigator: https://anaconda.org/channels/anaconda/packages/anaconda-navigator/overview
3. Open a terminal (bash or zshrc) and create a new environment: `conda create --name <your-env-name>`
4. Open Anaconda Navigator and install + open Juyter Notebook from your newly created environment
5. From the repository's base directory, open `./src/main.ipynb`
6. Run all the cells
